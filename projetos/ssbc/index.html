Sistema de Suporte à Base de Custos

Essa ferramenta, desenvolvida em Django, tem como objetivo auxiliar na interação com um banco de dados Postgres dedicado aos dados de endereçamento e custeio, principalmente energia, água e iluminação pública.

É um sistema de CRUD capaz de tratar e importar um grande volume de dados, através de arquivos CSV, para o banco de dados.
