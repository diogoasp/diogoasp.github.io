Análise de Desempenho Escolar

Um projeto Desenvolvido a pedido do Secretário de Educação Ciência e Tecnologia de Campos dos Goytacazes para analisar o desempenho escolar com base nos dados do CAED, Censo INEP, iEducar e o Censo Municipal.

Esse projeto é uma solução para rápida análise de um conjunto de dados de diferentes fontes e estruturas. Todos os filtros e informações em tela tem como objetivo responder questões levantadas pelo Secretário e sua equipe.

Sem dúvidas não se destaca pelo visual, mas foi projeto para atender necessidades específicas e com tempo muito curto de desenvolvimento.

Desenvolvimento

A primeira tela é simples, somente filtros básicos, onde é possível selecionar uma ou mais escolas para analisar.

Na segunda tela temos uma visão geral da escola selecionada (caso tenha selecionado uma lista, é alteranar entre as selecionadas com os botões de navegação ao lado do nome da escola).

Nessa tela a distribuição dos alunos de acordo com o conceito no CAED. Há também a média e o conceito geral da escola. Mais abaixo é possível ver a quantidade de alunos por conceito nas avaliações de Língua Portuguesa e Matemática. Por último, mas não menos importante, temos no canto inferior direito uma matriz com a distribuição por turmas (esse visual é passível de filtrar os demais para visualização por turmas). Nessa mesma matriz há um botão no canto superior direito, onde alterna-se o valor de referência, onde é possível ver a distribuição baseado no CAED, no Censo do INEP ou no Censo Municipal.

Na última tela temos uma visualização da relação de alunos com indicador visual do lado esquerdo para indicar se o aluno pode estar tendo dificuldades de aprendizagem e uma série de filtros a fim de atender às necessidades do secretário.
